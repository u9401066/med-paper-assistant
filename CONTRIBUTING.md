# 貢獻指南 (Contributing Guidelines)

感謝您有興趣為 Medical Paper Assistant 做出貢獻！

## 如何開始

1.  Fork 本專案。
2.  建立您的 Feature Branch (`git checkout -b feature/AmazingFeature`)。
3.  提交您的修改 (`git commit -m 'Add some AmazingFeature'`)。
4.  推送到 Branch (`git push origin feature/AmazingFeature`)。
5.  開啟 Pull Request。

## 開發規範

- **程式碼風格**: 請遵循 PEP 8。
- **語言**: 專案文件主要使用繁體中文，程式碼註解可使用英文或中文。
- **測試**: 請確保新增的功能包含適當的測試。

## 回報問題

如果您發現 Bug 或有新功能建議，請使用 GitHub Issues 回報。
