---
description: Perform statistical analysis on uploaded data.
---

1. **List Data**: List files in `data/` directory.
2. **Select Data**: Ask the user which file to analyze and what variables to check (Group, Outcome).
3. **Analyze**: Use `run_statistical_test` and `create_plot` tools to analyze the data.
4. **Report**: Summarize the findings and point to the generated figures in `results/`.
